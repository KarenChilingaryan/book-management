# Book Management API

This is a RESTful API for managing books and authors, built using NestJS, TypeScript, and MySQL. The API supports CRUD operations for books and authors, and includes JWT-based authentication.

## Features

- CRUD operations for books and authors
- JWT-based authentication
- Input validation and error handling
- Swagger documentation - baseUrl+/api/docs

## Tech Stack

- Node.js
- NestJS
- TypeScript
- MySQL
- TypeORM
- Passport.js
- JWT
- Swagger

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- Node.js (>= 14.x)
- Yarn
- MySQL

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/your-username/book-management.git
   cd book-management

2. install packages

  ```bash
  $ yarn install
  ```

## Running the app

```bash
$ yarn start

$ yarn start:dev
```

## Test

```bash
$ yarn test
